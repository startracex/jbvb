# label-input

label-input
组件用于显示一个被label包含的输入框,并将其两端对齐,在屏幕宽度不足时,呈现列式布局。

| 属性  | 类型   | 默认值        | 描述          |
| ----- | ------ | ------------- | ------------- |
| label | string | ""            | label的值     |
| type  | string | "text"        | input的类型   |
| name  | string | laebl or type | input的name   |
| pla   | string | ""            | input的提示   |
| def   | string | ""            | input的默认值 |
| value | string | def           | input的值     |
