# React

godown可以在大部分React系列框架中使用

## 安装

```bash
npm install godown
```

## 引入

以base-button为例

```jsx
import { BaseButton } from "godown/out/react";
```

## 使用

示例,显示一个 base-button

```jsx
export default () => {
  return (
    <BaseButton
      onClick={() => {
        alert("click");
      }}
    >
      Click me
    </BaseButton>
  );
};
```
